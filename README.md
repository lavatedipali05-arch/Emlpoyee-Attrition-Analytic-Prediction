📊 Employee Attrition Analytic Prediction

🚀 Project Overview

This project predicts whether an employee is likely to leave the company (attrition) using Machine Learning.
It also provides data insights and visualizations to understand employee behavior.

---

🎯 Features

- 🔮 Attrition Prediction (Yes/No)
- 📊 Interactive Dashboard using Streamlit
- 📈 Data Visualization (Job Satisfaction, Age, Salary, etc.)
- 🤖 Machine Learning Model (Pipeline)
- 💡 Easy-to-use UI

---

🛠️ Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

📌 Key Insights

- Employees with low job satisfaction are more likely to leave
- Salary and work environment play a major role in attrition
- Younger employees show higher attrition rate
